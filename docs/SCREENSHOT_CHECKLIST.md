## Instructions

1. Run the app locally: `npm run dev`
2. Navigate to each page listed above
3. Use browser full-page screenshot tool (or `Cmd+Shift+4` on Mac, `Snipping Tool` on Windows)
4. Save with the exact filename shown above to `docs/assets/screenshots/`
5. Check the box in this list
6. Screenshots will automatically appear in the README

**Recommended resolution:** 1440×900 or higher  
**Format:** PNG (lossless for UI screenshots)
